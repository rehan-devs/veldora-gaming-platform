> Where Dragons Play and Dreams Float Away ☁️✨

The fluffiest blockchain gaming platform in the sky! A whimsical, playful landing page featuring Veldora, the friendliest (and sleepiest) dragon mascot.

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn or pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/veldora-landing.git

# Navigate to the project
cd veldora-landing

# Install dependencies
npm install

# Run development server
npm run dev
